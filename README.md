# bd_lr1
ЛР1_БД__234_Вольников_Шиневский - Приложение на PyQT для визуализации данных

Десктопное приложение для анализа и визуализации данных на PyQt5.

**Возможности**\
Загрузка данных: CSV и Excel файлы с автоопределением кодировки\
Статистика: Описательная статистика и анализ пропусков\
Визуализация: Графики корреляции, тепловые карты, линейные графики\
Логирование: Журнал всех действий с временными метками\
Интуитивный интерфейс: Вкладки на русском языке\

**Установка:**\
bash\
pip install pyqt5 pandas numpy matplotlib seaborn\

**Использование:**\
Нажмите "Загрузить файл данных" на вкладке "Статистика"\
Выберите CSV или Excel файл\
Переключайтесь между вкладками для анализа данных\

**Вкладки:******\
1.Статистика - основная информация о данных\
2. Графики корреляции - парные диаграммы рассеяния\
3. Тепловая карта - матрица корреляций\
4. Линейные графики - графики по отдельным столбцам\
5. Ход работы - журнал действий\

Приложение автоматически определяет числовые столбцы и строит визуализации только для них.
